## 83 Remove Duplicates from Sorted List

## ğŸ“  Problem

Given the head of a **sorted linked list**, delete all duplicate nodes such that each element appears only once. Return the linked list in sorted order.

### Examples 1
![list1](https://github.com/user-attachments/assets/085012bd-a607-40a5-b06f-90ce9b08f957)

```
Input:  [1,1,2]
Output: [1,2]

```
### Examples 2
![list2](https://github.com/user-attachments/assets/3f07b29c-7e2e-4f73-b793-ee61931b0809)
```
Input: head = [1,1,2,3,3]
Output: [1,2,3]
```

### Constraints

* The number of nodes in the list is in the range [0, 300].
* -100 <= Node.val <= 100
* The list is guaranteed to be sorted in ascending order.
